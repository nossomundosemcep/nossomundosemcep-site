INSTRUÇÕES — site simples (HTML + CSS)
Arquivos incluídos:
  - index.html
  - styles.css
  - images/product1.svg
  - images/product2.svg
  - images/product3.svg

Como usar:
1) Descompacte o arquivo e envie todos os arquivos para a pasta pública da sua hospedagem
   (ex: public_html, www, htdocs) usando o painel da hospedagem ou FTP/SFTP.
2) Substitua os arquivos images/*.svg pelas suas fotos reais (mantendo os nomes ou atualizando as referências em index.html).
   - Para usar fotos reais, recomendo formatos JPG ou WEBP. Troque o atributo src das imgs para o caminho do arquivo.
3) Atualize os contatos (e‑mail e WhatsApp) no <head> e na seção Contato:
   - Substitua "contato@nossomundosemcep.com.br" pelo seu e‑mail real.
   - Substitua o link WhatsApp "https://wa.me/5511999990000" pelo seu número no formato internacional (ex: 5511987654321).
4) Se quiser alterar cores/brand, edite styles.css (variáveis no topo: --accent, --accent-dark, etc).
5) Para publicar: após os arquivos no servidor, acesse seu domínio (ex: https://www.seudominio.com.br).

Observações:
- O formulário de contato usa action="mailto:..." (abre o cliente de e‑mail do usuário). Se preferir, posso incluir um formulário que envie para um serviço (ex: Formspree) ou um script simples em PHP.
- Posso adaptar o site para múltiplas páginas ou criar um pequeno CMS (WordPress) no futuro se desejar.
